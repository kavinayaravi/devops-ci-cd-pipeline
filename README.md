AutoDeployX 🚀
Automated CI/CD Deployment Pipeline

📌 Overview
AutoDeployX is a DevOps project that automates the process of building, testing, and deploying applications using a CI/CD pipeline. Whenever code is pushed to the repository, the pipeline automatically builds the application and deploys it, reducing manual effort and speeding up delivery.

🛠 Technologies Used
Git & GitHub
Jenkins
Docker
Linux
Shell Scripting

⚙️ Workflow
Developer pushes code to GitHub
Jenkins triggers the CI/CD pipeline
Application is built and tested
Docker image is created
Application is deployed automatically

🚀 Features

Automated deployment pipeline
Faster and reliable software delivery
Reduced manual intervention
